關於我

我熱愛旅行，勇於走出舒適圈，這份愛讓我對於在國外獨自生活充滿信心。今年三月成功考取「華語導遊人員」、「日語領隊人員資格」。這些證書不僅是我的專業認證，更是總結我在語言、旅遊方面的里程碑。


不斷追求進步，今年參加勞動部的「嵌入式機器學習開發實作班」，深入學習Python程式語言，訓練AI，希望讓自己的視野擴展到更遙遠的未來。


此外，我也透過自學，積極學習 HTML 和 CSS ，此網頁為響應式網頁，能針對不同螢幕的大小而自動調整網頁圖文內容。


這一路走來，我深切體會到學習和成長的價值，我深信這些努力將為我開啟更多精彩的旅程。我期待能夠將我的多元專業帶到不同的地方，與人互動、交流，志同道合的夥伴們一起工作，建立公司團隊的成就。
